Optimized Route

Route optimization with WebScraping from Google Maps, extracting the time and distances of combinations of all routes. It returns the best route according to a linear programming calculation, considering weight 1 for time and weight 0.001 for distance. It also has the option of generating the best route using the Salesman algorithm. Both options plot the map with the trajectory of the best route found.

The tools used were:

  1. Python
  2. Folium
  3. Selenium
